In order to begin bookkeeping some setup is required.


Setup Facility
--------------

### 1\. Navigate to Setting -> Setup -> Facility.

### 2\. Click Add.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/6602037838c49dbc532ff97d_2024-03-25%2015_59_35-Facility.png)

### 3\. Enter Facility information.

### 4\. Click Save.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660203b15eab650cbdc24f74_2024-03-25%2016_06_45-Facility.png)

Setup Vendor  

---------------

### 1\. Navigate to Vendor -> Vendor Accounts.

### 2\. Click Add Vendor.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/66020488661187e9c5fea183_2024-03-25%2016_10_03-Vendor%20Accounts.png)

### 3\. Enter Vendor information.

This includes several categories of information:

*   Basic Information (required).
*   Address (required).
*   Contract (required)
*   Additional Information (optional).
*   Payment Information (required).
*   File
*   Insurance
*   Notes

### 4\. Click Save.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660204c9556032426f8d777e_2024-03-25%2016_11_21-Basic%20Information.png)

Setup Customer
--------------

### 1\. Navigate to Customer -> Customer Accounts.

### 2\. Click Add New Account.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660303919fb51f2ceb3dfbef_2024-03-26%2010_03_01-Customer%20Accounts.png)

### 3\. Enter Customer information.

This includes several categories of information:

*   Basic Information (required).
*   Address (required).
*   Contract (required).
*   Billing Rule (optional).
*   Billing Code Design (optional).
*   Dynamic Reference Field (optional).
*   Financial (required).
*   Ratings/Billing (optional).
*   Notes
*   File
*   Balance
*   View Mapping
*   View Change Code
*   Reference
*   Other
*   Attributes
*   Deals
*   Questionnaire

### 4\. Click Save.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660303a3b09bdcbfaffedada_2024-03-26%2010_03_47-Basic%20Information.png)
