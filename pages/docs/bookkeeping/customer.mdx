Setup Customer
--------------

### 1\. Navigate to Customer -> Customer Accounts.

### 2\. Click Add New Account.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660303919fb51f2ceb3dfbef_2024-03-26%2010_03_01-Customer%20Accounts.png)

### 3\. Enter Customer information.

This includes several categories of information:

*   Basic Information (required).
*   Address (required).
*   Contract (required).
*   Billing Rule (optional).
*   Billing Code Design (optional).
*   Dynamic Reference Field (optional).
*   Financial (required).
*   Ratings/Billing (optional).
*   Notes
*   File
*   Balance
*   View Mapping
*   View Change Code
*   Reference
*   Other
*   Attributes
*   Deals
*   Questionnaire

### 4\. Click Save.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660303a3b09bdcbfaffedada_2024-03-26%2010_03_47-Basic%20Information.png)

Edit Customer
-------------

### 1\. In the Customer list select a customer.

### 2\. A menu will appear, select Edit Account.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660c382842b97af542ad47aa_2024-04-02%2009_53_31-Customer%20Accounts.png)

### 3\. Use the navigation bar on the left to select the information you wish to edit.

![__wf_reserved_inherit](https://uploads-ssl.webflow.com/656a2fae21236d39b5805a4d/660c3998a536a3730234306c_2024-04-02%2009_59_33-Basic%20Information.png)

### 4\. Edit Customer information.

### 5\. Select Save.
